# Nomis BOT

- Auto farming
- Auto mission
- Support multi account
- Register Nomis [here](https://t.me/NomisAppBot/app?startapp=ref_Lndkhu-VNo)

### Installation

``git clone https://github.com/Galkurta/Nomis-BOT.git
``

``
cd Nomis-BOT
``

``
npm install
``

## Usage

### Edit data.txt 

``
nano data.txt
``

### Create screen

``
screen -S your_name_screen
``

### Run 

``
node main.js
``
